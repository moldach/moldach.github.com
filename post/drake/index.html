Getting Started with Drake in R

Matthew J. Oldach
Feb 5, 2019 1 min read

Recently I got interested in using the xaringan package for creating HTML based R Markdown documents because I could use the widgetframe package to embed htmlwidgets as responsive iframes (I had created some spinning 3D visulizations with pseudo-coloring I wanted to present interactively).

xaringan, it’s a R pacakges for creating slideshows with remark.js through R Markdown.

Around the same time I learnt about Drake (or “Data Frames in R for Make”) which is an amazing a time-saving reproducible build system for data scientist specifically tailored to R. The pacakge makes use of the futures package allowing the use of parallel computing on high-performance computing systems.

I recently gave a talk at the Douglas Mental Health University Institute talking about the advantages of Drake (and a few other interesting projects).
